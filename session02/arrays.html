1. Write a function that returns all people born in 1963. (loops not allowed!)

2. Write a function that returns a new array with persons. But the person objects should only contain the "surname" property. (loops not allowed!)

3. Write a function that accepts a string argument (surname) and returns a boolean value indicating if there is at least one person with the passed surname. (loops not allowed!)

4. Write a function that accepts a string argument and returns a boolean value indicating if the passed argument can be found in the surname of ALL persons. (loops not allowed!)

Hint 1: All four exercises can be solved by using the convenience functions of the Array API. (e.g. filter(), map(), some(), every(), etc.)

Hint 2: You can checkout the Git repository in order to execute the code locally to test your solutions.
